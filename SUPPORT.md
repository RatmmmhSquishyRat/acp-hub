# Support

## Getting help

1. **Docs first** — [README](README.md) (purpose, install, getting started, cheatsheet).
2. **CLI help** — `acp-hub --help` and `acp-hub <command> --help` are authoritative.
3. **Agent skill** (optional) — repository `.grok/skills/acp-hub/`, release
   archive `skills/acp-hub/`.
4. **Questions** — [GitHub Discussions](https://github.com/RatmmmhSquishyRat/acp-hub/discussions) (Q&A / Ideas).
5. **Bugs / features** — [Issues](https://github.com/RatmmmhSquishyRat/acp-hub/issues) with the right template.

## What this project does *not* provide

- Guaranteed response times or commercial SLA
- Support for third-party agents you register (omp, Codex, Cursor, Grok, …) beyond Hub wiring
- Remote multi-tenant hosting (Hub is local-first)

Sensitive reports: [SECURITY.md](SECURITY.md) only.
